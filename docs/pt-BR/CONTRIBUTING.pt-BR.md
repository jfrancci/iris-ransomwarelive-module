# Contributing

Thank you for contributing!

## Report Bugs

- Check existing issues first
- Include IRIS version, Python version, OS
- Provide logs and steps to reproduce

## Suggest Features

- Describe the enhancement clearly
- Explain use case and benefits
- Consider backwards compatibility

## Pull Request Process

1. Fork repository
2. Create feature branch
3. Make changes
4. Test thoroughly
5. Update documentation
6. Submit PR

## Coding Standards

- Follow PEP 8
- Use type hints
- Write docstrings
- Add comments for complex logic
- Keep functions small

## Testing

- Test on IRIS v2.4.22+
- Verify both manual and automatic modes
- Test with different ransomware groups
- Check logs for errors

See full guidelines in repository.
